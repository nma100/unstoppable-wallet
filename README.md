# Unstoppable Wallet
This application displays the assets of a user's wallet.

User's wallet address is obtained with “UNS Domain Resolution API”.

## Build

Install dependencies :
```
npm install
```

Create production build :
```
npm run build
```

Start development server :
```
npm start
```
